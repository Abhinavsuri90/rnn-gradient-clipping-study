# Assignment 26: Gradient Clipping in Deep Networks

This project contains a comprehensive Jupyter Notebook demonstrating the exploding gradient problem in Deep Recurrent Neural Networks (RNNs) and how to mitigate it using gradient norm clipping and LSTMs.

## 📦 Requirements

To run this project, you will need the following Python packages:
- `torch`
- `numpy`
- `pandas`
- `matplotlib`
- `seaborn`
- `tqdm`
- `jupyter`
- `ipykernel`

## 🚀 How to Run

1. **Create and activate a virtual environment:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. **Install the dependencies:**
   ```bash
   pip install torch numpy pandas matplotlib seaborn tqdm jupyter ipykernel
   ```

3. **Run the Notebook:**
   Launch the notebook server:
   ```bash
   jupyter notebook assignment.ipynb
   ```
   *Alternatively, if you are using an IDE like VS Code, simply open `assignment.ipynb`, set your Python kernel to the `venv` environment, and click "Run All".*
